# Bibliothèque Ésotérique

Site web dédié à la littérature ésotérique et aux savoirs mystiques. Ce site présente une collection d'ouvrages, de grimoires et de textes anciens liés à l'ésotérisme.

## Fonctionnalités

- Collection de grimoires et textes ésotériques
- Biographies d'auteurs mystiques
- Effets visuels cosmiques
- Interface interactive avec effets stellaires
- Curseur personnalisé avec effets de particules
- Design responsive

## Technologies Utilisées

- HTML5
- CSS3 (Animations et effets avancés)
- JavaScript (Effets visuels dynamiques)
- Canvas pour les générations d'images
